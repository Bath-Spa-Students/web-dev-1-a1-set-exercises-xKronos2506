Quick Introduction to HTML tags

HTML (HyperText Markup Language) is the standard programming language designed for web development, mainly webpages and web applications. HTML is used as the structure of a webpage and its content.

Here are following examples of HTML tags:

HTML Declaration Tag

This is a tag used to declare that the document is rendered as an HTML document. This is an example of the HTML declaration tag.

<!DOCTYPE html>

HTML tag

This is a tag used to represent the root of an HTML document, which specifies that the document within this tag is an HTML file.

<html>This is the html tag</html>

head tag

This is a tag used for containing metadata, which is data about the HTML document, such as its title, style, character set, etc.

<head>This is a head tag</head>

body tag

This is a tag used for containing all of the main structure of the HTML document, particularly the texts and images.

<body>This is a body tag</body>

heading tag

This is a tag used for displaying header text on different sizes. Within the heading tag, there are only 6 levels of heading.

<h1>This is a h1 tag</h1>

<h2>This is a h2 tag</h2>

<h3>This is a h3 tag</h3>

<h4>This is a h4 tag</h4>

<h5>This is a h5 tag</h5>
<h6>This is a h6 tag</h6>

Paragraph tag

This tag is used for displaying paragraph contents, mainly text.

<p>This is a paragraph tag</p>

This is a paragraph tag

List Tags

These tags are used to compose a list, either an ordered or unordered list.

Unordered: 

<ul>
    <li>This is an unordered list</li>
    <li>This is an unordered list</li>
    <li>This is an unordered list</li>
</ul>
  • This is an unordered list
  • This is an unordered list
  • This is an unordered list
Ordered: 

<ol>
    <li>This is an ordered list</li>
    <li>This is an ordered list</li>
    <li>This is an ordered list</li>
</ol>
  1. This is an ordered list
  2. This is an ordered list
  3. This is an ordered list

Anchor tag

This tag is used for creating hyperlinks to other web pages through text, which users can interact with by clicking it.

<a>This is an anchor tag</a>

Image Tag

This tag is used to embed an image within an HTML page.

<img src="imagelink.com" alt="image">

Div tags

These tags are used for marking a division within a certain block of code, for the purpose of editing them into a CSS stylesheet.

<div>This is a div tag</div>
I added a border, background color, and font color using the div tag

Quick introduction to CSS tags

CSS (Cascading Style Sheet) is a programming language designed for styling the HTML language.

Background Color

This tag is used for changing the background color of a specified tag.


.css-bg-color {
    background-color: red;
}

The background color of this text is red!

Text color

This tag is used for changing the text color of a specified tag.


.css-text-color {
    color: blue;
}

The color of this text is blue!

Text align

This tag is used for changing the text alignment of a specified tag.


.css-text-align {
    text-align: center;
}

The text is aligned at the center

Font Size

This tag is used to change the font size of a specified tag.


.css-font-size {
    font-size: 20px;
}

The text font size is now 20px!

Font Family

This tag is used to change the font family of a specified tag.


.css-font-family {
    font-family: Arial, Helvetica, sans-serif;
}

The text font family is now Arial!

Width

This tag is used to change the width of a specified tag.


.css-width {
    width: 400px;
    background-color: blue;
}

The following paragraph has a width of 400px

Margin

This tag is used to change the margin of a specified tag.


.css-margin {
    margin: 10px;
    background-color: blue;
}

This following paragraph has a margin of 10px

Height

This tag is used to change the height of a specified tag.


.css-height {
    height: 300px;
    background-color: blue;
}

The following paragraph has a height of 150px
